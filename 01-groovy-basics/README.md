# Part 1 — Groovy Basics

## 🎯 Goal
Get comfortable with the Groovy you will see in every Jenkinsfile: lists, maps, closures, strings, and control flow.

## 🧠 What you practise here
- `def` variables and types
- GStrings (`"... ${value} ..."`) vs plain strings
- Lists and the methods `each`, `findAll`, `collect`, `sum`
- Maps and how to read/loop over them
- Closures (little blocks of code you pass around)
- `if`, `switch`, ranges, and writing methods

---

## 📝 The 3 exercises

| # | File | What it tests |
|---|------|---------------|
| 1 | `exercise-1-collections.groovy`        | lists & maps |
| 2 | `exercise-2-closures.groovy`           | closures + `each`/`findAll`/`collect` |
| 3 | `exercise-3-strings-and-control.groovy`| GStrings, `if`/`switch`, methods |

Open each file, complete the `// TODO` lines, then run it:

```bash
groovy 01-groovy-basics/exercise-1-collections.groovy
```

> No Groovy installed yet? Paste the file into the [Groovy Web Console](https://gwc-experiment.appspot.com/).

Ready-made answers are in [`solutions/`](solutions).
